# 1С UI Monitor

Программа для мониторинга и логирования действий оператора в 1С через UI Automation API.

## Установка

```bash
pip install -r requirements.txt
```

## Запуск

```bash
python main.py
```

## Описание

Приложение использует UI Automation API для отслеживания элементов интерфейса 1С без доступа к внутренним ресурсам платформы. Аналогично работе Inspect.exe, но с автоматическим логированием.

## Структура проекта

- `main.py` - точка входа
- `gui/` - графический интерфейс (PyQt5)
- `monitor/` - модули мониторинга UI элементов
- `requirements.txt` - зависимости

## Возможности

- Мониторинг UI элементов 1С в реальном времени
- Логирование информации об элементах (тип, имя, значение, AutomationId, ClassName)
- Простой GUI для управления мониторингом
- Экспорт логов
